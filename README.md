# Sistema Financiero - Backend

Este directorio contiene el código del servidor (Node.js + Express).

## Estructura
- src/: Código fuente principal
- controllers/: Lógica de negocio y controladores
- models/: Modelos de datos
- routes/: Definición de rutas y endpoints
- utils/: Funciones y utilidades
- assets/: Imágenes, logos, etc.

## Configuración
- .env: Variables de entorno
- package.json: Dependencias y scripts
- Dockerfile: Configuración para contenedores

---

Recuerda inicializar el proyecto con `npm init` y agregar Express como dependencia principal.
